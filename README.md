# 📖 Project Overview

This repository contains multiple Flutter applications developed to explore different aspects of cross-platform mobile development.

The projects demonstrate Flutter's ability to build high-performance Android and iOS applications using a single codebase while integrating modern technologies such as Firebase Authentication, Cloud Firestore, SQLite, state management, and responsive UI design.

These applications showcase practical software engineering principles including reusable components, scalable architecture, offline-first functionality, and cloud synchronization.

---

# 📱 Applications Included

### 🧳 Smart Travel Journal

A travel diary application allowing users to record journeys, attach images, save memories, and synchronize travel logs using cloud storage.

---

### 🎂 Birthday Reminder

An application for managing birthdays with reminders, notifications, and local data persistence.

---

### 📖 Dictionary Application

A multilingual dictionary and translation application providing word meanings, translations, and common usage examples.

### 📌 Other Flutter Practice Projects

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/chaudharymuzamil03/Flutter-Projects-App.git
```

Navigate to the project

```bash
cd Flutter-Projects-App
```

Install dependencies

```bash
flutter pub get
```

Run

```bash
flutter run
```
